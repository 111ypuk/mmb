Проект Эклипс. Перспектива J2EE.
Импортируется в workspace и можно работать.

Сборка WAR
Создать директорию deploy в проекте.
WAR-файл собирается с помощью ant, его нужно установить, если ещё не установлен.
Для сборки проекта должен быть установлен на машине Apache Tomcat 7.0.
В файле build.xml нужно заменить путь к Apache Tomcat 7.0 на тот, где в системе реально установлен tomcat.
Для сборки надо в командной строке переместиться в директорию проекта, затем выполнить команду "ant export-war".
