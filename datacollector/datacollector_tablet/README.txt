От Эклипса отказываемся.
Проект AndroidStudio. Можно сразу открывать после загрузки из git.
Как проект подключается папка datacollector.

Для устойчивой работы AndroidStudio надо загрузить и установить приложения git (поискать в интернете "git download windows").
После открытия проекта надо настроить систему VCS. Выбрать тип vcs - git.
