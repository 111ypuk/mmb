Разработка приложения ведётся в IDE Eclipse.
Основной репозитарий вытаскивается на локальную машину разработчика.
Проект terminal можно импортировать в workspace Eclipse из директории git-base/mmb/terminal.
Проект terminal-tests можно импортировать в workspace Eclipse из директории git-base/mmb/terminal/tests.

Для соблюдения общих правил форматирования нужно подключать настройки:
Java -> Code Style -> Formatter -> Import...
Импорт настроек из файла git-base/mmb/terminal/formatter_settings.xml.
