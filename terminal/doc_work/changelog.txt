ChangeLog

------------- terminal-v1.5 -------------

Изменения:
1) Изменилась структура БД. Таблица Levels не используется.
   Все функции перекочевали в LevelPoints. Meta для импорта подправлена;
2) Добавился импорт данных с терминалов считывания штрих-кодов;
3) Подключение сканера штрих-кодов напрямую к планшету полностью удалено;
4) Введено понятие ScanPoint - точка, где считываются штрих-коды.
   Команды с двух дистанций приходят на одну ScanPoint, при этом
   у них разные LevelPoint. Дистанция определяется по номеру команды.
   Номер команды уникален внутри ММБ.
   
Доработки:
1) Реализован расчёт результатов.

------------- terminal-v1.4 -------------

Проект переведён на API-18, с поддержкой версий до API-7;

Доработки:
1) Добавлена работа со сканерами штрих-кодов;
2) Настройки сохраняются в кеше приложения, а не в БД;
2а) Путь к БД теперь не задаётся жёстко. Можно выбрать любой файл на устройстве;
2б) Можно импортировать файл с настройками с сайта (ID юзера, девайса, пароль);
3) Формы перевёрстаны под альбомную раскладку для удобства работы 
   с экранами 600 пикселов высотой;
4) Созданы все нужные условия для Eclipse -> Export... release-версии (keystore, proguard).


------------- terminal-v1.3 -------------

Доработки:
1) Форма поиска команды переделана в History.

2) Небольшие доработки интерфейса.
