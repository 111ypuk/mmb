b t SETIxx - изменить ID сканера
b t SETCxx - изменить  контрольную точку (CP)
c SETLx - вкл/выкл проверку длины ШК (длина берется из образца)
c SETNx - вкл/выкл фильтрацию ШК с буквами (N - буквы не разрешены)
b t SETPxxxxxxxxx - изменить образец ШК. Звездочка "*" в строке означает любой символ.
b t SETT - изменить текущее время

b GETS - получить текущие настройки логгера
b f GETL - получить полный лог сканирования
b f GETD - получить список отсканированных строк, не прошедших фильтрацию (некорректных)
b f GET#Lxxx - получить строку №xxx из файла с корректными сканами
b f GET#Dxxx - получить строку №xxx из файла с некорректными сканами
b DELLOG - удалить оба лог-файла
b END - вернуться в режим сканирования
