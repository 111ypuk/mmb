SETIxx - изменить ID сканера
SETCxx - изменить  контрольную точку (CP)
SETLx - вкл/выкл проверку длины ШК (длина берется из образца)
SETNx - вкл/выкл фильтрацию ШК с буквами (N - буквы не разрешены)
SETPxxxxxxxxx - изменить образец ШК. Звездочка "*" в строке означает любой символ.
SETT - изменить текущее время

GETS - получить текущие настройки логгера
GETL - получить полный лог сканирования
GETD - получить список отсканированных строк, не прошедших фильтрацию (некорректных)
GET#Lxxx - получить строку №xxx из файла с корректными сканами
GET#Dxxx - получить строку №xxx из файла с некорректными сканами
DELLOG - удалить оба лог-файла
END - вернуться в режим сканирования
